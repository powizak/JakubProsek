# Jakub Prošek - Osobní Webová Prezentace

Oficiální portfolio a osobní web Software Architekta a AI Integrátora Jakuba Proška.

## 🛠️ Technologie
Projekt je postaven na moderním stacku zaměřeném na maximální výkon (Lighthouse 100%) a čistotu kódu.

- **Vite** (Build tool & Dev server)
- **TypeScript** (Typed Logic)
- **SCSS** (Styling w/ variables & nesting)
- **Vanilla JS** (Žádný těžký framework jako React/Vue pro maximální rychlost)

## 🚀 Spuštění projektu

1. **Instalace závislostí:**
   ```bash
   npm install
   ```

2. **Spuštění lokálního serveru:**
   ```bash
   npm run dev
   ```

3. **Build pro produkci:**
   ```bash
   npm run build
   ```
   Výsledné soubory naleznete ve složce `dist/`.

## 🎨 Design System
- **Avant-Garde Minimalismus**
- **Dark Mode** (Onyx Black & Neon Green/Cyan accents)
- **Scrollytelling** navigace
- **Interaktivní prvky** (Custom cursor, hover effects)
